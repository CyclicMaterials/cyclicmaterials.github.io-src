# Flex

A minimalist [Pelican](http://blog.getpelican.com/) theme.

## Features

- Mobile First
- Responsive
- Semantic
- SEO Best Practices
- Open Graph
- Rich Snippets (JSON-LD)

## Integrations

- AddThis
- Disqus
- Google Analytics
- Google Tag Manager
- StatusCake
